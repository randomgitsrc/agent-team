## 经验总结

### Claude Code 调用策略
1. 先试试不加参数能否正常调用
2. 如果报错：
   - 先读取 $CLAUDEMODEL 环境变量
   - 如果有值 → 用 --model $CLAUDEMODEL
   - 如果为空或仍报错 → ~~用 --model combo-code~~ (已过时)

### TaskTree 组件选择
- NTree 的 @select 事件在浏览器自动化中无法触发
- 改用 NCollapse + 递归组件实现父子层级
- 点击事件改用普通的 @click

### 测试规范
- 浏览器自动化可能有缓存/延迟，必要时要重启服务
- 手动测试更可靠
