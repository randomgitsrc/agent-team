### 2. 多 Agent 设计讨论

| 议题 | 结论 |
|------|------|
| Telegram vs Discord | 3个 agent 用 Telegram 够用 |
| Channel 路由 | 需要配置 /spawn、/switch 命令 |
| 消息可见性 | subagent 对话需要透明可见 |
| 记忆共享 | memory 共享，session 独立 |
| Discord 互通 | 不，各互通平台独立 |
