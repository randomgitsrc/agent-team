## 💡 Web 工具使用经验

### Web 工具层级
```
Level 1: web_search  → 发现内容（URL）
Level 2: web_fetch  → 快速提取静态内容（首选）
Level 3: browser   → 完整浏览器自动化（兜底）
```

### web_fetch 限制
- 不支持代理配置
- 不支持 JS 渲染页面
- Firecrawl 不支持 X.com 等反爬站点

### 选择原则
- 能直接访问的网站 → web_fetch（如果不用代理）
- 需要代理访问的网站 → browser 工具
- 动态/JS 渲染页面 → browser 工具

### 触发规则
- 用户说"访问网站"、"打开网页" → 用 browser 工具
- 用户说"查一下"、"搜索一下" → 用 web_search + web_fetch

## 🔌 MCP 插件配置

### 安装
1. 克隆 `openclaw-mcp-adapter` 源码
2. `npm install` + `npm run build`
3. `openclaw plugins install`

### 配置
```json
{
  "plugins": {
    "entries": {
      "openclaw-mcp-adapter": {
        "enabled": true,
        "config": {
          "servers": [{
            "name": "github",
            "transport": "stdio",
            "command": "npx",
            "args": ["-y", "@modelcontextprotocol/server-github"],
            "env": { "GITHUB_PERSONAL_ACCESS_TOKEN": "ghp_xxx" }
          }]
        }
      }
    },
    "allow": ["openclaw-mcp-adapter"]
  }
}
```

### 状态
- ✅ Gateway 运行中
- ✅ 插件已加载
