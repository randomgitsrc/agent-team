## 🚨 高风险命令处理

### 事件
浏览器服务挂掉 → 直接执行 `openclaw gateway restart`

### 违反规则
`gate risky-cmd`：重启核心服务需人工确认

### 根因
把工具的错误提示（"Restart the OpenClaw gateway"）当成了执行许可，跳过了人工确认环节。

### 修复
1. 更新 AGENTS.md 明确此规则
2. 创建 `safe-exec.sh` 拦截
3. 确立流程：工具建议 ≠ 执行许可
