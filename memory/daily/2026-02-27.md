### 关键经验 💡

#### 1. OpenClaw Memory 配置路径
- **位置**: `agents.defaults.memorySearch`（不是顶层 `memorySearch`）
- **合法 provider**: `openai`, `gemini`, `voyage`, `mistral`, `local`
- **自定义端点**: 用 `provider: "openai"` + `remote.baseUrl` + `remote.apiKey`

#### 2. Embedding 模型选择
| 模型 | 状态 | 原因 |
|------|------|------|
| BCE-embedding-base_v1 | ❌ 失败 | 512 tokens 限制，OpenClaw chunk 超出 |
| Qwen3-Embedding-0.6B | ✅ 可用 | 无 token 限制，性价比高 |
| Qwen3-Embedding-8B | ✅ 最终选用 | 质量更好，Gitee 资源包付费 |
| Qwen3-VL-Embedding-8B | ❌ 不可用 | 名称错误，Gitee 未部署此模型 |

#### 3. 端点配置格式
```
# 正确
https://ai.gitee.com/v1          # 去掉 /embeddings 后缀
https://api.siliconflow.cn/v1    # 标准 OpenAI 兼容端点

# 错误
https://ai.gitee.com/v1/embeddings  # 404
```
