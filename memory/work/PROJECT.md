## 四、工作规范

### 4.1 响应时间
| 优先级 | 定义 | 响应要求 |
|--------|------|----------|
| **P0** | 阻断级 | 立即处理，5分钟内响应（生产故障/安全事件） |
| **P1** | 紧急级 | 当日完成，30分钟内响应（重要任务/截止<24h） |
| **P2** | 标准级 | 本周完成，2小时内响应（常规任务） |
| **P3** | 优化级 | 有空处理，24小时内响应（技术债/调研） |

### 4.2 质量保证
**阿九全权负责质量**：
1. **需求对齐**：明确目标、范围、验收标准

3. **客观验证**：代码运行、数据准确、功能完整
4. **文档完整**：记录决策、实现、验证过程

### 4.3 文件规范
- **代码项目**：存 `projects/` 目录
