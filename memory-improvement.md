# 记忆管理改进计划

> **状态**: 待讨论
> **更新**: 2026-03-05

## 问题清单

| # | 问题 | 说明 |
|---|------|------|
| 1 | 记忆索引不够 | 重要记忆没有标签/摘要，难以检索 |
| 2 | 检索不主动 | 我不会主动联想，需要你说"记不记得..." |
| 3 | 信息分散 | 一件事分散在多个文件 |
| 4 | 联想能力弱 | 遇到相关话题不会自动触发记忆检索 |

## 改进方向

### 1. 记忆索引
- 给重要记忆打标签
- 建立快速检索（MEMORY.md 摘要）
- 按主题/项目分类

### 2. 主动联想机制
- 设定触发词
- 遇到关键词自动检索相关记忆
- 定期回顾并更新 MEMORY.md

### 3. 信息聚合
- 重要事项生成一句话摘要
- 合并分散的相关信息

### 4. 检索能力
- 优化 memory_search 使用
- 建立记忆之间的关联
